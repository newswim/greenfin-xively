##### _About_

Greenfinity Garms is a Denton, Texas-based non-profit that wants to bring intelligent
Aquaponics into people's lives. For more info on this project, read the [TODO](./todo.md).

#### **Starting the app**

- Have Meteor installed (min. v1.3.1)
- `npm install`
- `meteor`
